limine áreas de sombra de sinal de uma vez por todas! A linha Deco utiliza um sistema de unidades para obter uma cobertura Wi-Fi completa em toda a casa. Com a avançada tecnologia Mesh, as unidades trabalham juntas para formar uma rede unificada com um único nome. Seus dispositivos alternam automaticamente entre os Decos enquanto você se move pela casa, dessa forma você não precisa se preocupar em se conectar em várias redes. O kit Deco M5 com três unidades é ideal para cobrir uma área de até 510m², enquanto o kit com duas unidades oferece Wi-Fi para uma área de até 350m². E se isso não for suficiente, basta adicionar mais Decos à sua rede a qualquer momento para aumentar a cobertura.
